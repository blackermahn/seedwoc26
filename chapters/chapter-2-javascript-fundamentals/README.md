# Chapter 2: JavaScript Fundamentals

## 📚 Overview

Learn the foundations of JavaScript - the programming language of the web. Master variables, data types, functions, and the DOM to start building interactive web experiences.

## 🎯 Learning Objectives

By the end of this chapter, you will:
- Understand JavaScript syntax and data types
- Write functions and understand scope
- Work with objects and arrays
- Master string manipulation
- Understand control flow (loops, conditionals)
- Interact with the DOM
- Debug JavaScript effectively
- Follow JavaScript best practices

## 📖 Curriculum

### Module 1: Syntax & Basics
- **Part 1**: Variables and Data Types
  - let, const, var differences
  - Primitive vs reference types
  - Type coercion
  
- **Part 2**: Operators & Control Flow
  - Arithmetic, logical, comparison operators
  - if/else conditionals
  - switch statements
  - Ternary operators
  
- **Part 3**: Loops & Iteration
  - for, while, do-while loops
  - for-of, for-in loops
  - break and continue

### Module 2: DOM Manipulation
- **Part 1**: Selecting & Querying Elements
  - getElementById, querySelector
  - CSS selectors
  - NodeLists vs HTMLCollections
  
- **Part 2**: Modifying the DOM
  - Creating elements
  - Changing properties and styles
  - Adding/removing classes
  
- **Part 3**: Event Handling
  - Click, submit, input events
  - Event listeners and handlers
  - Event delegation
