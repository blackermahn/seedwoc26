# Module 2: DOM Manipulation

## 📚 Module Overview

Learn how to interact with web pages. Understand the Document Object Model (DOM) and how to select, modify, and create HTML elements with JavaScript.

## 🎯 Learning Objectives

By completing this module, you will:
- Select DOM elements efficiently
- Modify HTML and CSS from JavaScript
- Create and delete elements dynamically
- Handle user interactions with events
- Build interactive web applications
- Understand event delegation
- Create dynamic user interfaces

## 📖 Module Outline

### Part 1: Selecting & Querying Elements
Learning to find elements on the page

### Part 2: Modifying the DOM
Changing HTML, CSS, and content

### Part 3: Event Handling
Responding to user actions

## ⏱️ Duration

- **Estimated time**: 2 weeks
- **Difficulty Level**: ⭐⭐ Intermediate

## 📝 Assignments

This module contains exercises covering:
1. Element selection methods

4. Element creation and removal
5. Event listener attachment
6. Event delegation patterns
7. Building interactive components
