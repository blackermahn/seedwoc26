# Chapter 4: Python Fundamentals

## 📚 Overview

Learn Python, one of the most beginner-friendly yet powerful programming languages. Python is used for everything from web development to data science and machine learning.

## 🎯 Learning Objectives

By the end of this chapter, you will:
- Understand Python syntax and philosophy
- Work with all Python data types effectively
- Write functions and modules
- Understand object-oriented programming
- Work with files and directories
- Use built-in libraries
- Debug Python code effectively
- Follow PEP 8 style guidelines

## 📦 Tools & Resources

**Installation:**
```bash
# Check Python version
python3 --version

# Create virtual environment
python3 -m venv venv

# Activate (Mac/Linux)
source venv/bin/activate

# Activate (Windows)
venv\\Scripts\\activate
```

## 💡 Pro Tips

1. **Use virtual environments**: Always isolate project dependencies
2. **Follow PEP 8**: Python community values clean code
3. **Use type hints**: Help catch bugs early (Python 3.5+)
4. **Test your code**: Use unittest or pytest
5. **Read the source**: Python code is often readable and learnable
