# Chapter 4, Module 1: Python Basics and Control Flow

## Module Overview

This module introduces Python programming fundamentals. You'll learn Python syntax, data types, variables, operators, and control flow structures - the essential foundations for all Python development.

## Learning Objectives

By completing this module, you will be able to:
- Set up Python development environment
- Understand and use Python data types effectively
- Declare and manipulate variables
- Use operators for calculations and comparisons
- Control program flow with conditionals
- Create loops for repetitive tasks
- Write clean, readable Python code
- Debug simple Python programs

## Curriculum Outline

### Part 1: Python Setup and Basics
- Installing Python and IDEs
- Running Python scripts
- Interactive Python REPL
- Python syntax and indentation
- Comments and documentation
- Data types overview: int, float, str, bool
- Variables and naming conventions
- Type conversion and type checking
- Duration: 2-2.5 hours
- Prerequisites: Computer basics, command line familiarity

### Part 2: Operators and Expressions
- Arithmetic operators (+, -, *, /, //, %, **)
- Comparison operators (==, !=, <, >, <=, >=)
- Logical operators (and, or, not)
- Assignment operators
- Operator precedence and associativity
- String concatenation and formatting
- Membership and identity operators
- Duration: 2-2.5 hours
- Prerequisites: Part 1

### Part 3: Conditional Statements
- If, elif, else structure
- Boolean logic and truthiness
- Nested conditionals
- Conditional expressions (ternary operator)
- Common conditional patterns
- Guard clauses
- Switch patterns (match/case in Python 3.10+)
- Duration: 2-3 hours
- Prerequisites: Part 1, Part 2

### Part 4: Loops and Iteration
- While loop
- For loop with range()
- Iterating over sequences
- Break and continue statements
- Else clause with loops
- Nested loops
- Loop patterns and best practices
- Duration: 2.5-3 hours
- Prerequisites: Part 3
