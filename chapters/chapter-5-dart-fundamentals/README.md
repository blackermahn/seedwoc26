# Chapter 5: Dart Fundamentals

## 📚 Overview

Master Dart, a modern programming language created by Google. Dart is the language behind Flutter for mobile development and is increasingly used for full-stack development.

## 🎯 Learning Objectives

By the end of this chapter, you will:
- Understand Dart syntax and features
- Work with Dart data types and collections
- Create functions and closures
- Use object-oriented programming in Dart
- Handle null safety concepts
- Work with asynchronous Dart
- Understand Dart package management
- Prepare for Flutter development

## 📖 Curriculum

### Module 1: Syntax & Basics
- **Part 1**: Installation and Setup
  - Dart SDK installation
  - IDE setup (VS Code with Dart extension)
  - Running Dart programs
  
- **Part 2**: Variables and Types
  - Dart type system
  - Null safety (late, required)
  - Collections and generics
  
- **Part 3**: Functions and Operators
  - Function declarations
  - Named parameters
  - Arrow functions

### Module 2: OOP in Dart
- **Part 1**: Classes and Objects
  - Class definition
  - Constructors
  - Getters and setters
  
- **Part 2**: Advanced Concepts
  - Inheritance and mixins
  - Abstract classes
  - Interfaces
  
- **Part 3**: Functional Dart
  - First-class functions
  - Async/await
  - Streams

## 📦 Tools & Resources

**Installation:**
```bash
# Install Dart SDK
brew install dart  # macOS
sudo apt-get install dart  # Ubuntu

# Check installation
dart --version

# Create new project
dart create my_project
```

## 💡 Pro Tips

1. **Embrace null safety**: It prevents bugs before runtime
2. **Use pub for packages**: It's Dart's package manager
3. **Learn async early**: Essential for modern Dart
4. **Test with unit tests**: High test coverage is valued
5. **Follow effective Dart**: Read the style guides
