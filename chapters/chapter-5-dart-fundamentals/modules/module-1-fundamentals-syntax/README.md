# Chapter 5, Module 1: Dart Fundamentals and Syntax

## Module Overview

This module introduces Dart programming language fundamentals. Created by Google and used primarily for Flutter app development, Dart combines familiar syntax with powerful features for building scalable applications.

## Learning Objectives

By completing this module, you will be able to:
- Set up Dart development environment
- Understand Dart syntax and type system
- Work with variables and data types
- Use operators and expressions
- Control program flow with conditionals
- Create loops for repetitive tasks
- Write clean, idiomatic Dart code
- Understand null safety in Dart

## Curriculum Outline

### Part 1: Dart Setup and Language Basics
- Installing Dart SDK
- Dart development environment options
- Your first Dart program
- Variables and naming conventions
- Data types: int, double, String, bool
- Type inference with var
- Constants (final and const)
- Null safety concepts
- Duration: 2-2.5 hours
- Prerequisites: Computer basics, command line familiarity

### Part 2: Operators and Control Flow
- Arithmetic operators
- Comparison operators
- Logical operators (&&, ||, !)
- Assignment operators
- Conditional operators (ternary)
- If, else if, else statements
- Switch-case statements
- Conditional expressions
- Duration: 2-2.5 hours
- Prerequisites: Part 1

### Part 3: Collections and Iteration
- Lists and list operations
- Sets and set operations
- Maps for key-value storage
- Iterating with for loops
- Iterating with forEach
- Spread operator
- Collection comprehensions
- Duration: 2.5-3 hours
- Prerequisites: Part 2

### Part 4: Functions and Methods
- Function declaration and calling
- Parameters (positional and named)
- Default parameters
- Return types and return statements
- Arrow functions
- Optional and required parameters
- Higher-order functions
- Closures in Dart
- Duration: 2.5-3 hours
- Prerequisites: Part 1, Part 3
