# Chapter 5, Module 2: Object-Oriented Programming in Dart

## Module Overview

This module covers object-oriented programming in Dart. You'll learn classes, inheritance, mixins, and other OOP concepts that are essential for building scalable applications with Dart and Flutter.

## Learning Objectives

By completing this module, you will be able to:
- Create and use classes in Dart
- Understand constructors and object initialization
- Implement inheritance effectively
- Use mixins for code reuse
- Understand abstract classes and interfaces
- Implement polymorphism
- Use getters and setters
- Design class hierarchies
- Apply SOLID principles in Dart

## Curriculum Outline

### Part 1: Classes and Object Basics
- Class declaration and structure
- Properties and methods
- Constructors and initialization
- Named constructors
- Factory constructors
- Getters and setters
- This keyword
- Property initialization shorthand
- Duration: 2.5-3 hours
- Prerequisites: Chapter 5, Module 1

### Part 2: Inheritance and Polymorphism
- Class inheritance with extends
- Super keyword and parent methods
- Method overriding
- Operator overriding
- Polymorphism concepts
- Protected and private members
- Abstract classes
- Duration: 2.5-3 hours
- Prerequisites: Part 1

### Part 3: Mixins and Interfaces
- Mixin syntax and usage
- Creating custom mixins
- Mixin order and resolution
- Interfaces in Dart (implicit via implements)
- Multiple implementation
- When to use mixins vs inheritance
- Real-world mixin patterns
- Duration: 2-2.5 hours
- Prerequisites: Part 1, Part 2

### Part 4: Advanced OOP Concepts
- Enumerations (enums)
- Sealed classes
- Extension methods
- Static members
- Design patterns in Dart
- Effective OOP principles
- Functional programming in Dart
- Duration: 2.5-3 hours
- Prerequisites: Part 1, Part 2, Part 3
