# Chapter 6, Module 1: React Fundamentals and JSX

## Module Overview

This module introduces React, the popular JavaScript library for building user interfaces. You'll learn the core concepts including components, JSX, state, and props that form the foundation of React development.

## Learning Objectives

By completing this module, you will be able to:
- Understand React's component model and philosophy
- Write React components using JSX
- Work with functional and class components
- Manage component state with hooks
- Understand component lifecycle
- Pass data between components with props
- Render lists and conditionals
- Handle basic styling in React
- Set up a React development environment

## Curriculum Outline

### Part 1: React Setup and JSX Basics
- What is React and why use it?
- Setting up React project (Create React App, Vite)
- JSX syntax and compilation
- Functional components
- Component structure and best practices
- Rendering components to the DOM
- babel and JSX transformation
- Duration: 2-2.5 hours
- Prerequisites: JavaScript Fundamentals (Chapter 2-3)

### Part 2: Components, Props, and Events
- Functional components and reusability
- Props and passing data to components
- Props destructuring
- Default props
- Event handling in React
- Binding and context in event handlers
- Event object and synthetic events
- List rendering with .map()
- Keys in lists
- Duration: 2.5-3 hours
- Prerequisites: Part 1

### Part 3: State and Hooks
- React Hooks introduction
- useState hook for state management
- useEffect hook for side effects
- Hook dependency arrays
- Custom hooks basics
- Multiple state variables
- State updates and re-renders
- Conditional rendering
- Duration: 3-3.5 hours
- Prerequisites: Part 2

### Part 4: Component Composition and Styling
- Component composition patterns
- Lifting state up
- Container vs presentational components
- Styling approaches (inline, CSS, CSS-in-JS)
- Conditional classes
- CSS Modules
- Introduction to pre-processors (SASS/SCSS)
- Accessibility in React
- Duration: 2.5-3 hours
- Prerequisites: Part 1, Part 3

## Pro Tips

✨ **Tip 1**: React is a library, not a framework - it focuses on the UI layer

✨ **Tip 2**: Functional components with hooks are the modern way - embrace them

✨ **Tip 3**: JSX looks like HTML but it's JavaScript - understand the transformation

✨ **Tip 4**: Props are read-only - state is for mutable values

✨ **Tip 5**: Component composition is more powerful than inheritance - favor composition
