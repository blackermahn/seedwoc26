## Interview Preparation

### Technical Interview Components

#### Coding Interviews
**Focus Areas by Role:**
- **Web Dev:** JavaScript, HTML/CSS, APIs, databases
- **Backend:** System design, databases, algorithms
- **ML:** Statistics, Python, ML concepts, math
- **DevOps:** Linux, networking, infrastructure

**Preparation Time:** 8-12 weeks of daily practice

**Resources:**
- LeetCode (algorithms)
- HackerRank (coding)
- System Design Primer (architecture)
- Project Euler (math)

#### System Design Interviews
**Key Topics:**
- Scalability
- Database design
- Caching strategies
- Load balancing
- Microservices
- API design

**Practice:** Design 2-3 real systems (Twitter, Uber, Netflix)

#### Behavioral Interview

**Common Questions:**
- "Tell me about yourself"
- "Describe a challenging project"
- "How do you handle disagreement?"
- "What's your biggest achievement?"

**Framework:** STAR (Situation, Task, Action, Result)

### Interview Timeline

**3 Months Before:**
- Master fundamentals
- Start system design practice
- Develop portfolio projects

**1 Month Before:**
- Mock interviews
- Practice coding under time pressure
- Research company and role

**1 Week Before:**
- Light review
- Prepare questions to ask
- Get good sleep

**Day Of:**
- Arrive early
- Be confident
- Ask clarifying questions

## Building Your Portfolio

### Portfolio Essentials

**Quality over Quantity:**
- 3-5 complex projects better than 20 trivial ones
- Show problem-solving, not just coding
- Demonstrate learning journey

### Project Selection

**Choose Projects That Demonstrate:**
1. Full-stack capability (if applicable)
2. Problem-solving skills
3. Code quality and best practices
4. Scalability thinking
5. Collaboration (open source contribution)

### Portfolio Presentation

**Should Include:**
- Live demos (hosted projects)
- GitHub repositories (clean, well-documented)
- README with clear descriptions
- Technical blog posts
- YouTube videos (optional)

### Recommended Portfolio Projects

**Level 1 - Beginner:**
1. Todo application
2. Weather application
3. Personal portfolio website
4. Calculator app
5. Quiz application

**Level 2 - Intermediate:**
1. E-commerce website
2. Social media clone
3. Real-time chat application
4. Blog with CMS
5. Project management tool

**Level 3 - Advanced:**
1. Microservices architecture project
2. Machine learning application
3. Real-time collaborative tool
4. Mobile app (iOS/Android equivalent)
5. Open-source contribution

## Freelance vs Employment

### Freelancing Path

**Advantages:**
- Flexibility
- Diverse projects
- Higher hourly rates possible
- Work from anywhere

**Disadvantages:**
- Income variability
- No benefits
- Client acquisition overhead
- Isolation

**Platforms:**
- Upwork
- FreeCodeCamp
- Toptal
- Gun.io
- Arc.dev

**Starting Freelance:**
1. Build portfolio first
2. Start on Upwork/Fiverr
3. Move to specialized platforms
4. Build own client base
5. Network and referrals

## Negotiation Tips

### Salary Negotiation

**Research:**
- Glassdoor, Levels.fyi
- PayScale, Blind
- Geographic cost of living

**Negotiation Tactics:**
1. Research market rate
2. Get offer in writing
3. Don't disclose previous salary
4. Counter thoughtfully
5. Consider total compensation
6. Be professional

**What to Negotiate:**
- Base salary
- Bonus structure
- Equity/stock options
- Remote work flexibility
- Professional development budget
- Vacation time

### Benefits Beyond Salary

- Health insurance quality
- 401(k) matching
- Remote work policy
- Professional development
- Flexible hours
- Stock options/equity

## Next Steps After SeedWOC

1. **Pick Your Specialization** (1-2 weeks)
   - Use this guide to decide
   - Research job market
   - Assess your interests

2. **Build Foundational Skills** (8-12 weeks)
   - Complete relevant chapters
   - Do provided exercises
   - Build simple projects

3. **Develop Deep Expertise** (12-16 weeks)
   - Specialized courses
   - Build complex projects
   - Study real-world systems

4. **Build Portfolio** (8-12 weeks)
   - 3-5 significant projects
   - Document everything
   - Get feedback

5. **Interview Preparation** (8-12 weeks)
   - Algorithm practice
   - System design
   - Behavioral interviews
   - Mock interviews

6. **Job Search** (4-12 weeks)
   - Apply to companies
   - Network
   - Negotiate offers
   - Start your career!
